##检查状态
###目标
学习如何检查版本库的状态  
##01 检查版本库的状态
使用git status 命令检查当前版本库的状态  
**执行代码**  

`$ git status`  
**命令行输出**  

`$ git status`  
` # On branch master`  
` nothing to commit (working directory clean)`  

上述显示出了当前没有要提交的任务。这说明版本库保存的是工作目录的当前状态(最新状态)，没有什么突出的改变需要被记录下来。  

我们将使用git status命令继续监视版本库与工作目录的状态变化。
